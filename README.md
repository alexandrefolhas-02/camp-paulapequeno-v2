# Paula Pequeno Elite Camp V2

Projeto React + Vite para a Landing Page do Elite Camp.

## Stack
- React 19
- Vite
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Wouter

## Deploy
O deploy está configurado para a Vercel como projeto estático. 

1. Conecte o repositório na Vercel.
2. Certifique-se de que o framework preset seja `Vite`.
3. Adicione as variáveis de ambiente necessárias (veja `.env.example`).
4. O build command é `npm run build` e a pasta de saída é `dist`.
5. O `vercel.json` garante o roteamento do SPA para `index.html`.

## Como rodar localmente
1. `npm install`
2. `npm run dev`
